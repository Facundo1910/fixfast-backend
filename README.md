# fixfast-backend

Backend del proyecto FixFast configurado con Spring Boot y Java 17.

## Requisitos
- Java 17 (JDK)
- Maven 3.9+

## Comandos útiles
- `mvn spring-boot:run` para levantar el servicio localmente en `http://localhost:8080`.
- `mvn test` para ejecutar los tests.

## Endpoints iniciales
- `GET /health` verifica el estado básico de la aplicación.
